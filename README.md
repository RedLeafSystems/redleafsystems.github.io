# RedLeaf Systems — Static Site

This repository contains a modern, dark-themed static site scaffold for RedLeaf Systems.

Preview locally

PowerShell (Windows):
```powershell
# from repository root
python -m http.server 8000; Start-Process "http://localhost:8000"
```

Or with Node.js installed:
```powershell
npx serve -l 8000 .
```

Files of interest
- `index.html` — main page
- `assets/css/styles.css` — styles
- `assets/js/main.js` — small interactions
